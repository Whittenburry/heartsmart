# HeartSmart

A heart-healthy recipe adaptation MVP built as a showcase for John Whittenburg.

## Overview
HeartSmart allows users to paste a recipe URL, extracts the culinary data using `schema.org` / `JSON-LD`, and rebuilds the recipe to have fewer saturated fats and lower sodium using Google's Gemini AI. It provides an Instacart integration link to immediately shop for the adapted ingredients.

## Tech Stack
* **Frontend**: Vue 3 (Composition API), Vite, TypeScript, Tailwind CSS, Pinia, Vue Router
* **Backend**: Node.js, Fastify, TypeScript, Cheerio, Google GenAI SDK

## Setup & Running Locally

1. **Install Dependencies**
   ```bash
   cd client
   npm install
   
   cd ../server
   npm install
   ```

2. **Environment Variables**
   Create a `.env` file in the `server` directory:
   ```env
   PORT=3000
   GEMINI_API_KEY=your_gemini_key_here
   ```
   *Note: If `GEMINI_API_KEY` is omitted, the backend will automatically return mock adaptation data so you can still demo the UI flow.*

3. **Start Development Servers**
   In one terminal (Backend):
   ```bash
   cd server
   npm run dev
   ```
   
   In another terminal (Frontend):
   ```bash
   cd client
   npm run dev
   ```

4. **View App**
   Open `http://localhost:5173` in your browser.
